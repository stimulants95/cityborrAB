# Read Skills Before Starting Work

Before beginning any web design or development work, read all 10 skill files.

## Steps

1. Read all skill files from C:\Users\josef\.claude\skills\

2. Pipeline order:
   - anti-ai-patterns (read FIRST - non-negotiable rules)
   - project-intake - Client interview (4 phases)
   - design-system-generator - Colors, typography, spacing
   - component-builder - UI components
   - layout-breaker - Break monotonous layouts
   - typography-copy-polish - Text and hierarchy
   - image-media-integration - Creative image placement
   - animation-micro-interactions - Custom animations only
   - quality-gate - QA audit
   - final-polish-performance - Performance, SEO, launch

3. Check for design-brief.md in the project root.

4. Proceed with work, following anti-ai-patterns at all times.
